# AWS Security Lab

Projeto focado em estudos práticos de Cloud Security utilizando AWS.

---

## Objetivos

- Aprender fundamentos de segurança em nuvem
- Praticar IAM, CloudTrail e VPC
- Desenvolver conhecimento em ambientes AWS
- Criar portfólio para área de Cybersecurity

---

## Tecnologias Utilizadas

- AWS
- IAM
- CloudTrail
- Python
- GitHub

---

## Laboratórios

### IAM Security
- Criação de usuários
- Políticas de acesso
- MFA

### Monitoramento
- CloudTrail
- Logs
- Auditoria
