# AWS Security Lab

Laboratório prático focado em Cloud Security utilizando serviços AWS, automação e boas práticas de segurança em nuvem.

---

# Objetivo

Este projeto foi criado com o objetivo de desenvolver habilidades práticas em:

- Cloud Security
- Infraestrutura AWS
- Monitoramento e auditoria
- Hardening
- IAM e controle de acesso
- Automação com Python
- Boas práticas de segurança em ambientes cloud

Além disso, o projeto faz parte da minha jornada de transição para a área de Cybersecurity e Cloud Computing.

# Estrutura dos Laboratórios

## IAM Security Lab
Práticas relacionadas ao gerenciamento seguro de identidades e permissões.

### Conteúdos
- Criação de usuários IAM
- Políticas de acesso
- MFA (Multi-Factor Authentication)
- Princípio do menor privilégio (Least Privilege)

---

## Monitoring & Auditing Lab
Monitoramento e rastreamento de atividades no ambiente AWS.

### Conteúdos
- CloudTrail
- Logs de auditoria
- Monitoramento de eventos
- Rastreamento de atividades

---

## Networking & VPC Lab
Fundamentos de redes e isolamento em cloud.

### Conteúdos
- Criação de VPC
- Security Groups
- Regras de acesso
- Conceitos de rede em AWS
